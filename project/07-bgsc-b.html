项目简介

北广视彩平台采用与商户合作的模式铺设终端设备，通过C端线上公众号向彩民提供线下自助购彩的服务， 同时面向B端使用微信小程序来进行订单展示、佣金结算等服务。

B端小程序的用户有三种角色： ①业务员 ②合作方 ③终端商户，各个角色的相关功能如下：

项目技术
  • 前端使用微信小程序开发技术
  • 后端使用spring boot + dubbo + zookeeper微服务化
  • 数据库使用mysql，缓存使用redis
  • 使用gogs + maven + jenkins + docker + rancher等进行项目管理
项目职责

本人主要负责除“业务员管理”、“消息公告”外的其他功能开发及测试
