项目简介

我要结婚网致力于帮助新人完成从“美好婚礼”到享受生活的理想过程，以用户为核心，整合用户最为关注的信用、 品质、服务等环节存在的各种问题，以提供权威、真实、安全、贴心的全方位服务，努力打造全国最大的结婚交流服 务平台。

我要结婚平台主要包含PC、M站和管理后台三大模块产品线，目前各产品已迭代完成第二个版本，平台的主要功能如下：
项目技术
  • 项目基于Spring Cloud 全家桶微服务架构，使用Eureka作为注册中心
  • Redis缓存 + MySQL存储 + MyBatis Plus框架
  • 灵活运用的Spring AOP + Quartz任务调度
  • Web前端使用最新最流行的Beetl页面引擎，性能比Jsp更好
  • Web使用 Spring Redis Session解决分布式会话问题
  • Nginx + 集群Eureka配合部署
  • Gitlab + Maven的高效项目管理
备注：平台基于业务划分了User Service（用户服务）、Content Service（内容服务）、Common Service（公共服务，包括短信、CMS、城市）、Common Util（公共工具）、 Web（PC端展示模块）、WxService（微信小程序服务）等工程，以达到服务解耦的目的。
项目职责

本人主要负责整个“用户服务”、“公共服务”的相关功能设计及开发（包括数据库表的设计），以及“公共工具”的一些基础封装，解决项目开发中遇到的一些棘手问题。
