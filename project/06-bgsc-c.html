项目简介

北广视彩平台采用与商户合作的模式铺设终端设备，通过C端线上公众号向彩民提供线下自助购彩的服务，实现了无人值守自助售彩的目标。 平台的业务视图如下：

该公众号面向C端用户，对接福彩中心系统，支持双色球、3D、七乐彩三大彩种的售卖，提供了微信支付和农行微信支付的支付渠道， 后端集成云打印服务，用户下单后即可通知设备自动打印；若用户中奖，则也会通过公众号及时下发中奖推送消息。系统大致模块如图所示：

项目技术
  • 采用前后端分离模式开发，集成微信、支付宝支付
  • 前端使用vue框架及微信生态，token鉴权机制
  • 后端使用spring boot + dubbo + zookeeper微服务化
  • 数据库使用mysql，缓存使用redis及redisson分布式锁
  • 使用gogs + maven + jenkins + docker + rancher等进行项目管理
项目职责

本人主要负责了福彩3D和七乐彩投注下单、积分中心、体验券、支付方式切换等模块的相关功能开发及测试
