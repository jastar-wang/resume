项目简介

本系统作为运营后台管理系统，主要面向公司高层领导、管理、财务、客服、运营，通过权限模块进行划分，防止越级操作。

通过本系统，可以向领导直观的展示实时的销量数据、用户数据、设备数据等，财务人员可以方便的进行资金对账、门店的余 额管理及商户的佣金结算，客服人员可以根据用户的反馈来解决用户订单的相关问题，运营人员可以管理门店及设备铺设的相关数据。

项目技术
  • 采用前后端分离模式开发
  • 前端使用vue + element ui框架，token鉴权机制
  • 后端使用spring boot + dubbo + zookeeper微服务化
  • 数据库使用mysql，缓存使用redis及redisson分布式锁
  • 使用gogs + maven + jenkins + docker + rancher等进行项目管理
项目职责

本人主要负责用户管理（*）、商户管理（站点报备）、设备管理（*）、财务管理（打印机押 金，门店余额，兑奖列表）、业务数据视图等模块的功能开发与测试
